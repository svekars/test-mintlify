Get Started

Quickstart

Start building with the API in under 5 minutes

​
Get Your API Key

First, you’ll need an API key to authenticate your requests.
1

Sign up for an account

Go to dashboard.example.com and create your account.
2

Navigate to API Keys

In your dashboard, click on Settings → API Keys.
3

Create a new key

Click Create API Key, give it a name, and copy the key securely.
Store your API key securely. Never share it publicly or commit it to version control.

​
Make Your First Request

Let’s make your first API call to retrieve a list of users.
Copy
curl https://api.example.com/v1/users \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

​
Expected Response

Copy
{
  "data": [
    {
      "id": "usr_123abc",
      "email": "user@example.com",
      "name": "John Doe",
      "created_at": "2024-01-15T10:30:00Z"
    }
  ],
  "has_more": false,
  "total_count": 1
}

​
Create a User

Now let’s create a new user with a POST request.
Copy
curl -X POST https://api.example.com/v1/users \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "newuser@example.com",
    "name": "Jane Smith"
  }'

​
Expected Response

Copy
{
  "id": "usr_456def",
  "email": "newuser@example.com",
  "name": "Jane Smith",
  "created_at": "2024-01-15T11:30:00Z",
  "updated_at": "2024-01-15T11:30:00Z"
}
Success! You’ve created your first user via the API.

​
Install SDKs (Optional)

For easier integration, install one of our official SDKs:
  • Python
  • Node.js
  • Ruby
Copy
pip install example-api
Copy
from example_api import Client

client = Client(api_key="YOUR_API_KEY")
users = client.users.list()
